# Effectory Design System — Claude Code plugin

Bouw prototypes, mockups en component-demo's met de Effectory design-system componenten, tokens en iconen.

## Installatie

In Claude Code (één keer):

```
/plugin marketplace add jenteinsing123/effectory-design-documentation
/plugin install effectory-design-system
```

Daarna kun je in elk project zeggen: *"Bouw een prototype voor een settings-pagina"* — de skill activeert automatisch en levert een prototype dat de Effectory design-system regels volgt.

## Wat zit erin

- **Skill**: `effectory-design-system` — instructies en regels voor het bouwen van prototypes
- **Design-system bestanden**: `tokens.css`, `foundation.css`, `components.css`, `icons.js`, `serve.py` en alle iconen — worden bij het eerste gebruik in je werkdirectory gekopieerd

## Updates

Wanneer er een nieuwe versie verschijnt:

```
/plugin marketplace update effectory-design-plugins
/plugin install effectory-design-system
```
