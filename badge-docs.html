Badge

Merged into Tag. Effectory has one label component (the styleguide's Eff Tag) — a “badge” is simply a Tag at the default medium radius.

Completed = a Tag with the default (medium) radius.

There is no separate Badge component. Use Tag and pick a semantic background (info, brand, warning, negative, highlight, positive); leave the radius at medium for the compact status-marker look. Survey/report status has its own higher-level Survey Status component.
